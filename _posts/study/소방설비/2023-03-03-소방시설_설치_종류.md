# 2. 경보설비  

별도의 설명이 없으면 면적(m²)은 연면적을 말함.  

- 설치조건
  - 시각경보기 : 자탐설비를 설치하는 곳에 한해 아래 표에 따라 설치
  - 가스누설경보기 : 가스시설이 설치된 곳에 한해 아래 표에 따라 설치
  - 자동화재속보설비 : 아래 표에 따라 설치하나, 방재실 등 화재수신기가 설치된 장소에 감시인이 24시간 근무하는 경우 설치 제외할 수 있음
  - 누전경보기 : 계약전류 100A 이상인 곳에는  설치
  - 화재알림설비 : 판매시설 중 전통시장에 설치
- 면제조건
  - 단독경보형 감지기 : 자탐설비/화재알림설비를 적합하게 설치한 경우
  - 비상경보설비 : 자탐설비/화재알림설비를 적합하게 설치한 경우, 단독경보형 감지기를 2개 이상의 단독경보형 감지기와 연동하여 설치한 경우
  - 자탐설비 : 자탐설비의 (감지/수신/경보)기능과 성능을 가진 화재알림설비/스프링클러설비/물분무등 소화설비를 적합하게 설치한 경우
  - 화재알림설비 : 자탐설비를 적합하게 설치한 경우
  - 자동화재속보설비 : 화재알림설비를 적합하게 설치한 경우
  - 비상방송설비 : 자탐설비/비상경보설비와 같은 수준 이상의 음향을 발하는 장치를 적합하게 설치한 경우
  - 누전경보기 : 아크경보기/지락차단장치를 설치한 경우
