## 1. 전열

1. 전열의 개요
   - 전기에너지를 열 에너지로 변환
   - 변환방법 (가열 원리)에 따른 구분
      - 저항가열 : 저항에서 발생하는 줄열
      - 유도가열 : 전자유도에 의한 와전류 열
      - 유전가열 : 고주파 자계에 의한 분자 마찰열
      - 적외선 가열 : 적외선의 방사 열전달
      - 전자빔 가열
      - 레이저 가열
2. 전기가열의 특징
   - 매우 높은 온도를 얻을 수 있음
   - 내부 가열 가능
   - 열효율이 높음
   - 노기제어가 쉬움
   - 방사열의 이용 가능
   - 온도 및 가열시간의 제어가 쉽다
   - 제품의 품질이 향상됨
   - 환경을 오염시키지 않음

## 2. 전열의 계산

1. 용어
   1. 열량
      - 열량 $Q=mc\Delta T$ [J]
         - m [kg] : 물질 질량
         - c [J/kg°C] : 비열
         - $\Delta T$ [°C] : 온도 변화량
   2. 단위환산
      - 1 [cal] = 4.19 [J]
         - 1 [J] = 0.24 [cal]
      - 1 [kW h] = 3600 [kJ] = 860 [kcal]
      - 1 [BTU] = 0.252 [kcal]
   3. 비열 : 물체 1kg을 온도 1°C 상승시키는데 필요한 열량
   4. 열용량 : 물체의 온도를 1°C 상승시키는데 필요한 열량
   5. 온도상승
      - $\Delta T = \frac{Q}{C} = \frac{열량}{열용량}$
   6. 융해
      - 고체가 액체로 되는 현상
      - 융해 중 공급된 열은 온도상승이 아닌 상변화에 소비
      - 물의 융해열 : 80 [kcal/kg]
   7. 기화
      - 액체가 기체가 되는 현상
      - 물의 기화열 : 539 [kcal/kg]
2. 열전달
   1. 전도 : 고체 내에서의 열 전달
      - 열전도율 :  $q = -kA\frac{dT}{dx}$
         - q : 열류 [W]
         - k : 열 전도율 [W/m°C]
         - A : 단면적 [m²]
         - x : 두께 [m]
   2. 대류 : 유체 내에서 분자가 열에너지를 가지고 이동
      - 
   3. 복사 : 고온도의 물체에서 저온도의 물체로 전자기파의 형태로 방사
      - 
3. 열회로와 옴의 법칙
   - 양단 온도차가 크고, 단면적이 넓고, 길이가 짧을 수록 크다.
   - $I = \lambda \times \frac{l}{S} \times \Delta T$
   - $I = \frac{\Delta T}{R}$
      - $R = \frac{1}{\lambda} \times \frac{l}{S}$
      - I : 열류 [W]
      - S : 단면적 [m²]
      - l : 길이 [m]
      - λ : 열전도율 [W/m °C]
      - R : 열저항 [°C/W]
4. 열회로-전기회로 대응관계

   | 열회로 | 기호, 단위 | 전기회로 | 기호, 단위 |
   | --- | --- | --- | --- |
   | 온도차 | θ (ΔT) [°C] | 전위차 | V [V] |
   | 열류 | I [W] | 전류 | I [A] |
   | 열저항 | R [°C/W] | 전기저항 | R [Ω] |
   | 열전도율 | λ [W/m°C] | 도전율 | σ [℧/m] |
   | 열저항률 | ρ [m °C/W] | 저항률 | ρ [Ω m] |
   | 열량 | Q [J] | 전하량 | Q [C] |
   | 열용량 | C [J/°C] | 정전용량 | C [F] |

5. 소요열량, 소요전력량 계산
   1. 소요 열량
      - $Q = mc(T_2 -T_1) [\mathrm{kcal}] = 0.24Pt [\mathrm{kJ}] = 0.24I^2Rt$
   2. 소요 전력량
      - $Pt = \frac{mc(T_2 -T_1)}{860 \eta} [\mathrm{kWh}]$
   3. 전열기의 소요 용량
      - $P = \frac{4.18 mc(T_2 -T_1)}{\eta} [\mathrm{kW}]$
   4. 난방기 용량
      - $P = \frac{4.18 \times 1.24 \times V \times 0.24(T_2 -T_1)}{\eta} [\mathrm{kW}]$
         - 공기의 비중 : 1.24 [kg/m³]
         - 공기의 비열 : 0.24 [kcal/kg°C]
6. 전열선의 설계 계산
   - 전력 : P = EI = E²/R = I²R
   - 저항 : $R = \rho \frac{l}{A} = \rho \frac{4l}{\pi d^2}$  (A : 단면적)
   - 표면 전력밀도 : $W_d = \frac{P}{S} = \frac{I^2R}{\pi d l} =  \frac{4 \rho I^2}{\pi ^2 d^3}$
      - $\therefore W_d \propto \frac{1}{d^3}$
   - 선지름 $d = \sqrt[3]{\frac{4 \rho I^2}{\pi ^2 W_d}}$

## 3. 전기가열의 방식

1. 저항가열
   1. 직접저항 가열
      - 피열물 자체에 직접 직류전류나 상용주파수를 흐르게 하여, 줄열에 의해 발열시키는 방법
         - 흑연화로 : 탄소소재의 흑연화
         - 카보런덤로 : SiC 생성
         - 카바이드로 : CaO + 3C → CaC₂ + CO
   2. 간접저항 가열
      - 발열체에서 열을 발생시킨 다음 피열물로 전달 (방사, 대류, 전도)
         - 발열체로 : 노벽에 발열체 설치. 탄화규소를 사용하면 1500°C까지 가열 가능
         - 크립톨로 : 전극간에 설치된 탄소입자를 발열체로 사용. 1800°C까지 가열 가능
         - 염욕로 : 전극간 용융염을 발열체로 하는 로. 1300°C까지 가열 가능
2. 아크가열
   - 아크열을 이용하여 가열함
   - 전극으로는 인조흑연, 천연흑연을 이용
3. 유도가열
   - 가열원리 : 교번자계중에 놓인 유도성 물체의 와전류와 히스테리시스손에 의한 가열
   - 전원 : 교류전원만 사용 가능
      - 저주파 유도로 : 상용주파(60Hz) 사용
      - 고주파 유도로 : 5~20kHz 의 고주파 사용
   - 특징
      - 피가열물 내에서 직접 열을 발생시킬 수 있으며 별도의 열원 필요 없음
      - 표면층만 가열할 수 있음
      - 피가열물의 필요한 부분만 선택적으로 가열 가능
      - 온도제어가 정확하고 쉽다
      - 용해로의 자동 교반작용으로 양질의 제품을 얻을 수 있음
   - 응용
      - 반도체 정련 (단결정 제조)
      - 금속의 표면가열 (표면담금질, 표면처리, 국부가열)
   - 유도가열용 전원
      - 고주파 전동발전기
      - 불꽃 간극식 고주파 발생장치
      - 진공관 발진기
4. 유전가열
   - 교번전계 중 절연성 피열물에 생기는 유전체 손실에 의한 가열
   - 전원 : 교류전원, 1~100MHz
   - 유전체손 P
      - $P=VI_R = VIcos\theta = I_c\tan\delta$
         - $\delta = 90° - \theta$
      - $I_c = \omega CV = 2\pi fCV$
         - $\therefore P = 2\pi f CV^2 \tan \delta$
   - 평판 전극의 정전용량 C
      - $C = \frac{\epsilon_o \epsilon_s S}{d}$
   - 단위 체적당 유전체손 P0
      - $P_0 =\frac{P}{Sd} = 2\pi f V^2 \tan \delta  \frac{\epsilon_o \epsilon_s }{d^2} = 2\pi f \epsilon_o \epsilon_s E^2 \tan \delta$$  $
         - $\because E=V/d$
         -
   - 특징
      - 열이 유전체손에 의해 피열물 자체에 발생
      - 온도 상승속도가 빠르고, 속도를 임의로 제어할 수 있음
      - 전원이 끊어지면 가열이 즉시 멈춤. 주위 물체에 저축된 열에 의한 과열이 없음
      - 표면의 소손, 균열이 없음
      - 전효율이 낮음
         - 고주파발진기의 효율(5~60%)과, 회로손실 반영
      - 설비비가 고가임
      - 피열물의 기하학적 형상에 따라 가열 정도가 달라져서 균일 가열이 곤란함
      - 장치를 차폐하지 않으면 전파 누설에 의해 통ㅅ딘장애 발생
   - 응용
      - 목재 접착 (5~10MHz)
      - 목재 건조 (2~5MHz)
      - 기타 : 고무의 유화, 약품과 농어산물의 건조, 비닐막의 접착 등
   - 유도가열과의 비교

         | 항목 | 유전가열 | 유도가열 |
         | --- | --- | --- |
         | 가열 원리 | 유전체손 | 와류손, 히스테리시스손 |
         | 적용 대상 | 절연체 (유전체) | 금속(도체), 반도체 |
         | 전원 | 교류 1~200MHz | 저주파 : 60Hz, 고주파 5~20kHz |

5. 전기로
   1. 저항로
      1. 직접 가열식
         - 피열물에 직접 통전시켜 가열. 생산 제품(용도)에 따라 흑연화로, 카바이드로, 카보런덤로, 유리용융로, 알루미늄 전해로 등
      2. 간접 가열식
         - 발열체로 : Ni-Cr 등의 금속발열체 또는 SiC 등 비금속발열체에 통전가열
         - 염욕로 :  NaCl, KCl 등의 용융염에 직접 통전가열. 열처리에 많이 사용함
         - 크리프톨로 : 탄소와 점토의 혼합에 통전시켜, 탄소입자를 발열체로 사용
   2. 아크로
      1. (저압) 직접식 : 피열물을 아크 전극으로 사용.
      2. (저압) 간접식 : 전극간 아크에서 발생하는 열을 복사에 의해 피열물로 전달
         - 
      3. 고압 아크로
         - 
   3. 유도로
      1. 직접식 : 도전성 피열물에 직접 전류를 유기
      2. 간접식 : 피열물을 흑연도가니에 넣어서 흑연을 유도가열
6. 적외선가열
   - 원리 : 적외선전구에 의한 방사열(복사)로 가열
   - 특징
      - 도장 등의 표면건조에 적당함
      - 건조기 구조가 간단함 → 조작이 간단함
      - 설비비, 유지비가 저렴함
      - 설치장소가 컴팩트함
         - 연료 손실이 적고 작업시간이 단축됨
      - 건조 재료의 감시가 용이하고 청결하며 안전함

## 4. 전열 재료

1. 발열체의 구비조건
   - 내열성이 클 것
   - 내식성이 클 것
   - 적당한 고유저항값을 가지고, 저항의 온도계수가 양수이며 작을 것
   - 연성과 전성이 풍부하고, 가공이 용이할 것
   - 선팽창 계수가 작을 것
2. 발열체 종류
   - 금속 발열체 : Ni-Cr 합금선, Fe-Cr 합금선

      | 종류 | 고유저항 at 20°C <br> [μΩ cm] | 최고사용온도 [°C] | 특성 |
      | --- | --- | --- | --- |
      | Ni-Cr 제1종 | 104 ± 5 | 1100 | 가공성 양호. 고온에서 내열성, 내가스성이 강함. 고온 전기로용 |
      | Ni-Cr 제2종 | 110 ± 6 | 900 | 가공성 양호. 800~900°C 부근에서 사용하는 전기로 및 전열기에 적합 |
      | Fe-Cr 제1종 | 140 ± 7 | 1200 | 가공성 나쁨, 사용온도가 높으나 고온 강도는 떨어짐 |
      | Fe-Cr 제2종 | 120 ± 5 | 1100 | Fe-Cr 제1종에 비해 가공 용이함 |
   - 비금속 발열체 : 대표적으로 탄화규소 (SiC)를 주성분으로 하는 탄화규소질 발열체는 1400°C에서 장시간 사용가능
3. 전기로의 전극
   - 구비조건
      - 전기전도율 클 것
      - 열전도율 작을 것
      - 고온에 견딜 수 있고, 고온에서 기계적
   - 주로 사용되는 전극의 고유저항 [Ω cm]
      - 인조 흑연전극 : 0.5~1.2 × 10⁻³
      - 천연 흑연전극 : 3.0~7.6 × 10⁻³
      - 무정형 탄소전극 : 5~8  × 10⁻³
      - 고급 천연 전극 : 0.9~3.3  × 10⁻³

## 5. 온도 측정

1. 접촉식
   - 측정방법 : 온도 검출단을 물체 내부나 표면에 붙여서 검출단 온도를 측정대상 온도와 일치시켜 측정
   - 액체봉입 유리 온도계
   - 압력 온도계
   - 저항 온도계
      - 온도 변화에 따라 고유저항이 직선적으로 변화하는 것을 이용
      - 백금선, 니켈선 등의 금속선, 서미스터 등의 반도체 이용
         - 텅스텐 : 저항 온도계수가 작으므로 사용하기 어려움
   - 열전 온도계
      - 서로 다른 두 종류의 금속 또는 반도체의 접합점에서, 온도차에 의해 기전력이 발생 (제벡효과)
      - 종류와 측정범위

         | 열전대 | 사용범위 | 1회사용온도 |
         | --- | --- | --- |
         | Pt - Pt Rh (백금로듐) | 0~1400 | 1600 |
         | 크로멜-알루멜 | -200~1000 | 1100 |
         | 철-콘스탄탄 | -200~700 | 900 |
         | 구리-콘스탄탄 | -200~400 | 600 |
2. 비접촉식
   - 측정방법 : 측정 대상에서 나오는 방사 에너지로 온도 측정
   - 광고온도계
      - 플랑크의 복사 법칙을 이용
         - $W_\lambda = C_1 \lambda^{-5}(e^{C_2 / \lambda T}-1)^{-1}$
         - 단위표면적에서, 단위 파장폭당, 파장이 가지는 복사 에너지
      - 방사온도계에 비해 정도가 높음
      - 피측온물의 크기가 d=0.1mm
   - 방사온도계
      - 스테판 볼츠만 법칙 이용
         - $W = \sigma T^4$
      - 측정 대상의 방사율에 의해 온도 지시값이 다르므로 보정  필요
      - 온도 직독 가능
      - 피측온물에서 떨어진 위치에서 온도 기록 가능

## 6. 전기용접

1. 방전용접
   1. 원리 : 용접하려는 금속 모재와 용접용 전극 사이의 방전 열을 이용
   2. 탄소방전 용접
      - 주로 같은종류의 철 합금을 용접
      - 직류전원 사용
      - 용접물을 양극에, 탄소전극을 음극으로 사용
      - 가스용접에 비해 빠르고 경제적
   3. 원자 수소 용접
      - 경금속, 구리, 구리합금, 스테인리스강 등의 용접에 이용
   4. 불활성 가스 용접
      - 텅스텐 전극과 모재 사이에 아크방전을 발생시켜 용접
      - 아크 주위에 아르곤이나 헬륨 등의 불활성가스 피복을 이용해 용접부의 산화 방지
      - 용재 불필요
      - 알루미늄, 마그네슘, 스테인리스강, 기타 특수강 등의 방전용접에 적용
   5. 방전 용접기 (아크용접기)
      - 방전을 안정하게 지속시키기 위해 전압이 수하특성을 가져야 함. (저항변화시에도 전압 변동 X)
      - 방전 용접의 작업전압 : 20~35V
      - 용접용 전원 최고전압
         - DC : 50~70V
         - AC : 70~100V
2. 저항용접
   1. 원리 : 용접하고자 하는 두 금속 모재의 접촉부에 대전류 통전시켜 접촉저항으로 용융
   2. 겹치기 저항용접
      - 점용접 (스팟용접)
      - 프로젝션 용접
      - 심 용접
   3. 맞대기 저합용접
      - 업셋 용접
      - 플래시 용접
      - 충격 용접
3. 용접후 검사
   1. 비파괴 검사
      - 자기 검사
      - 방사선 투과시험 (X선, 감마선)
      - 초음파 시험
      - 육안검사
   2. 파괴 검사
      - 충격시험
      - 부식시험

## 7. 열펌프
1. 열펌프 개요
   - 외부의 기계적 일에너지를 이용해, 저온 물체의 열에너지를 고온 물체로 보내는 장치
2. 열펌프의 에너지 이동
   - 에너지 보존 법칙에 의해 : $Q_H = Q_L + W$
   - 성능계수 (C.O.P) : $\mathrm{COP} = \frac{Q_H}{W}$
  
## 8. 각종 효과

1. 표피효과
   - 도체에 흐르는 전류의 주파수가 높아질 수록 전류가 표면으로 집중되는 현상
   - 표면열처리에 이용
2. 제벡효과
   - 서로 다른 두 종류의 금속선을 접합하고 접합점의 온도를 다르게 하면 기전력이 발
   - 열전대 → 온도 측정
3. 톰슨효과
   - 동일 금속 도선의 두 점 간에 온도차가 있을 때, 전류가 흐르면 흡열이나 발열 반응
4. 펠티에 효과
   - 서로 다른 두 종류의 금속선을 접합하고 전류를 흘리면 접합점에서 흡열이나 발열 반응
   - 펠티에 소자 → 냉각
5. 핀치 효과
   - 유동성 도체에 전류 통전시 도체 중심방향으로 전자력이 작용하여 면적이 줄어드는 현상
   - 도체 면적 감소 → 저항 증가 → 전류 감소 → 인력 감소 → 도체 면적 복귀
